# Bilibili AI Video - Frontend

## 快速启动

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```

## 项目结构
```
src/
  ├── pages/
  │   └── index/
  │       ├── index.tsx       # 主页面
  │       └── index.config.ts  # 页面配置
  ├── app.config.ts           # 应用配置
  ├── app.tsx                 # 应用入口
  └── index.html              # HTML模板
```
